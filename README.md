## Automated Security Test Framework (Python)

A Python-based automation tool to validate network security compliance against predefined security baselines.

### Features
- Automated port scanning
- HTTP security header validation
- SSL/TLS configuration checks
- Generates test execution reports

### Tools & Technologies
- Python
- Socket Programming
- Requests Library
- SSL/TLS

### Outcome
- Reduced manual security testing time by ~40%
- Improved visibility of configuration-level vulnerabilities
